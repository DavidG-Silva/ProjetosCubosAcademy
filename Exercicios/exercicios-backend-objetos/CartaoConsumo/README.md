# Cartao de Consumo

- Crie um **objeto** que represente o *cartão de consumo* de um cliente. Deve ter:
    - Nome do cliente;
    - Idade do cliente;
    - Produtos consumidos.


- Cada produto pode ter:
    - Nome do produto;
    - Preço unitario (em centavos);
    - Quantidade comprada

Pode inventar os dados. Coloque pelo menos 3 produtos.
